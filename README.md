# Conversando por Voz com ChatGPT e Whisper 🎙️🤖

Este projeto utiliza **Whisper (OpenAI)** para transcrição de áudio e o **ChatGPT** para gerar respostas inteligentes.  
As respostas são convertidas em voz com **gTTS (Google Text-to-Speech)**.

## 🚀 Tecnologias
- Python
- Whisper (OpenAI)
- ChatGPT API
- gTTS
